# EJS ToDo APP

## Project Setup
- Clone the repository
  - `git clone https://github.com/kmadhav907/EJS-App.git`
- Install the dependencies  
  - `npm install`
- Run the App
  - `npm run server`
  - Open the Browser and go to localhost:3000
  
 ### Various Routes
  - / -> To main ToDo List
  - /about -> To about Page
  - /work -> To Work To Do list
